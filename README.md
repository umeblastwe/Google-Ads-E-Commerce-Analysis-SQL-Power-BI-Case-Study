# Google-Ads-E-Commerce-Analysis-SQL-Power-BI-Case-Study
Google Ads analysis using SQL &amp; Power BI on 750k+ rows. Found a major checkout "leak" (667k carts vs 15k sales), a 21.34% mobile success rate, and 10 PM peak buying hours. This case study provides a roadmap to fix UX friction and increase ROAS through targeted, seasonal ad spending. Perfect for optimizing e-commerce performance.
